  ⚠️ Partially Implemented / Placeholders
   * Visual "Hexagon" Stats Chart: RadarChart.tsx is currently in "Stability Mode" (a placeholder).
     This was a strategic choice to prevent runtime crashes caused by unstable SVG libraries on
     Android 14/15.
   * Shadow Soldiers: "Shadow Extraction" is implemented as a reward for daily streaks (Igris at 3
     days, Beru at 30 days, etc.), but the Interactive Shadow Soldiers 2.0 (assigning them to skill
     trees) is not yet active.

  ❌ Not Yet Implemented
   * Detailed Quest View: Currently, tapping a quest toggles completion; a dedicated detail/log view
     is not yet built.
   * System Inventory & Item Drops: There is no UI for item drops (Potions, Crystals) or inventory
     management.
   * "Gate" Exploration (Dungeons): The visual dungeon map and floor-clearing mechanics are not in
     the current build.
   * "The Sovereign's Voice" (TTS): Text-to-speech for system announcements is not yet integrated.
   * Supabase Cloud Sync: The app currently relies entirely on local AsyncStorage.
